# NuGet Alternatives to SunamoDebugIO

This document lists popular NuGet packages that provide similar functionality to SunamoDebugIO.

## Overview

I/O debugging utilities

## Alternative Packages

### Serilog
- **NuGet**: Serilog
- **Purpose**: Structured logging
- **Key Features**: File sinks, structured events, performance

### System.Diagnostics.Trace
- **NuGet**: System.Diagnostics.TraceSource
- **Purpose**: Built-in tracing
- **Key Features**: Trace listeners, event sources

### NLog
- **NuGet**: NLog
- **Purpose**: Flexible logging
- **Key Features**: Multiple targets, filtering, async logging

## Comparison Notes

Serilog for modern structured logging. NLog for configuration flexibility.

## Choosing an Alternative

Consider these alternatives based on your specific needs:
- **Serilog**: Structured logging
- **System.Diagnostics.Trace**: Built-in tracing
- **NLog**: Flexible logging
